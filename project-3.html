絶対フォント感

街中や写真の中で素敵なフォントを見つけた時、画像からフォントを検索したくなるが、そうしたツールが無料で公開されていないので自作した. Domain adaptationとCNNを用いた半教師あり学習モデルでひらがなのフォント推定モデルを構築,正答率8割を記録.

Project Overview

年賀状の文字入りの画像データに自分の文字を入れる際にフォントが分からず,無料で日本語のフォント推定できるものがなくて困ったので自作した. フォント認識の先行研究として、文字認識をせずにフォント推定するDeep Convolutional Neural Network(CNN)による手法で、80%以上の正答率で フォントを推定できるモデルが公開されている. 彼らは実世界データと生成データのdomainの違いや,光源,視点の違いによる多様なデータに対応するためにDomain Adapted CNNという自己教師学習モデルを開発している. 今回はこれを参考にしつて大量生成したひらがなデータと,ボカシなどによるData Augmentationと,先行研究を軽量化したモデルを用いて,ひらがなのフォント推定モデルを開発した. また、最適化アルゴリズムとしては本来SGDが用いられていたが、今回は Adamを採用した.

まとめ:Domain Adapted CNN + Data Augmentation → フォント推定!

結果,Adamで収束するまで学習したものが最もtest時の性能が高く、 83%の精度でフォントを正しく推定することができた. また、正解データでも不正解データでもtestデータにblurなど変形を かけた際の推定結果に変化が見られなかったことから、Data augmentationによってモデルがロバストになっていると確認された. 計算資源が豊富でモデルやデータセットのサイズを気にしないなら、学習するフォントの数を10000以上に増やせるので、大量のフォントで学習して推定時には上位３つを出してその中で人間が見比べるなどすればより性能が高く実用的になるだろう。
